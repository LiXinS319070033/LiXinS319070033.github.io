使用sass时，报错TypeError: this.getResolve is not a function at Object.loader

（在学习慕课网小米商城的项目5-1NavHeader组件(1)课程时）

原因：最开始在项目前期安装的sass的版本有问题，后来安装视频里的版本之后成功了。

切记：只更换sass-loader的版本不更换node-sass版本是不行的，必须两者都更换版本

解决方案：

切记：安装成功后要重启项目，否则会报出node的错误，只要重启一下项目就没node错误了。
