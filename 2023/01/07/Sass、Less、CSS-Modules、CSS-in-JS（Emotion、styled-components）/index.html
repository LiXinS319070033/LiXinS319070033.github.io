6三种运算

数字运算；字符运算；颜色运算

数字运算：

加：数值带单位时单位需要相同

减：数值带单位时单位需要相同；若做减法运算的是变量不是数值时，减号“-”前后必须有空格，才可以识别出

“-”不是变量的一部分

注意：带单位的数字与不带单位的数字可相加，结果是一个带单位的数字，因为带单位的数值我们可以把它单做一个数字处理，但带单位的数值与不带单位的数值是不可以相减的

乘：一个带单位，一个不带单位

除：”/“在css中已经作为一种符号使用了，例如字体属性编写font: Arial 12/1.5em;，若”/“两侧为数值时，需要加括号()，若”/“两侧至少一个是变量，不需要()括号

“/“作为除号有三种情况：
1.数值被小括号()包含；
2.数值是另外一个数学表达式的一部分；
3.数值存储在一个变量中或者函数的返回值；

css中有两种字符串类型：有引号字符号；无引号字符号

占位符：

混合宏：

用“@mixin”定义一个混合宏，用“@include”调用一个混合宏，可带参数，多个参数也可以，参数也可以设置默认值

总结和比较：

方法 声明方式 调用方式
继承 .class @extend
占位符 %placeholder @extend
混合宏 @mixin @include

继承和占位符的比较：
需要保留基类，只使用@extend实现；不需要保留基类，使用@extend配合%placeholder实现

继承和混合宏的异同点：
都能实现代码块的重用，极大提高开发效率；
继承的使用一般没有代码的冗余，而混合宏有代码冗余；
继承不能传递参数，而混合宏可以

9.导入

像某个js文件引入其他js文件一样，sass支持导入其他sass文件，使用@import最后会编译成一个单独的css文件，例：@import default.scss。在sass中引入其他sass文件时，可以忽略后缀名，例：@import default

在css和sass中都可以使用@import，但在编译的时候sass会将@import进来的sass文件合并进来只生成一个文件。但若在sass文件中导入css文件，那么效果和普通的css导入一样，不会被合并到编译后的文件中，还是以@import的形式存在
