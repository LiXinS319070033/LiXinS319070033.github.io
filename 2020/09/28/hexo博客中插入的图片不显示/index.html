图片不显示是个头疼的问题，试了好久才成功，我的解决过程是：

1、在cmd终端调到你自己的hexo的文件夹路径下，安装一个图片路径转换的插件

npm install https://github.com/CodeFalling/hexo-asset-image --save

但是这个插件的内容需要修改，不然可能会出问题

2、打开/node_modules/hexo-asset-image/index.js，将内容更换为下面的代码

3、打开hexo文件夹下的_config.yml文件，修改下述内容

post_asset_folder: true

4、将需要引用的本地图片存放在与文章名相同的文件夹中

_posts文件夹下有名为111.md的文件和一个名为111的文件夹，文件夹内存放着.md内使用的图片222.png

5、图片路径问题（至关重要）

我试了好多图片路径都没能实现，最终是这种相对路径的方式成功的实现了

./111/222.png
