​ 我目前博客的名字叫archer，可以在hexo官网的主题里面找到,点击Archer,整个安装过程比较简单，我是看到b站的这个视频了解到的，up主叫吃饱睡觉的猫，Makedown的语法与hexo的主题修改，其实在这之前我试过比如yilia-plus和melody这两个主题，但是可能是其中有一步git pull的原因，最终都没有更换成功，而archer这个主题没有那一步，后来可以更换成功了。hexo官网的主题很多，都是开源的，但是简易程度都不一样，喜欢哪个可以试一下。

hexo更换主题后进行个人博客设置图片时不显示

​ 在上一篇更换hexo主题后将里面信息改为自己的，后来发现添加的图片不能显示，是__图片保存的路径不对的原因__，图片都保存在在\themes\archer\source下面，根据自己实际情况保存。别忘了使用hexo g -d命令后才算部署成功。

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: true
    tags: true
