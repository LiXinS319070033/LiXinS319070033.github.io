hexo点击博客后标题不显示
​ 电脑修好后重新配置了hexo博客的环境，但是发现点击博客后标题不显示，如图片样子 想想应该是上传时的编码问题，因为上传标题乱码了， 目前没有找到通过乱码的解决方法，但是找到了另一个解决方法，就是在每次执行上传的操作hexo clean、hexo g、hexo d的地方，采用 hexo n "你的标题名称" 例：hexo n "Hello world" ，最终发现在每个.md文本最前面加上这句 便可以在博客中显示标题了。

npm i hexo-generator-json-content --save
